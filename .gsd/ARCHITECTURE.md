## Data Flow

1. User authenticates via Google OAuth or local login.
2. The Next.js client component makes `fetch` / `axios` calls to local `/api/*` endpoints.
3. The server API routes run business logic (e.g. generating quizzes, saving history) and query the MongoDB database via Mongoose.
4. AI integrations (e.g., Groq SDK, Google Generative AI) process prompts and return content which is then served back to the user interface.
